Nama Project : Kala Senja Canteen

Description Project : Projek ini dibuat untuk memudahkan kita dalam membeli makanan dan minuman. Kala Senja Canteen merupakan kantin umum yang menyediakan makanan dan minuman dengan cepat dan praktis. Kantin ini sudah menggunakan teknologi 4.0. Meskipun kantin ini termasuk kategori modern, namun kantin ini menyediakan makanan dan minuman tradisional, sehingga pembeli bisa bebas memilih menu apa yang mereka suka.
